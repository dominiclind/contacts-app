
## Getting started

Install dependencies

`yarn|npm install`

## Running app

### `npm|yarn start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000)

### `npm|yarn test`

Runs all test!

### `npm run|yarn build`

Builds the app for production to the `build` folder.<br>
